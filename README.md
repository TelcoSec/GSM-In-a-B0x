# GSM In A B0x



## Portable GSM Stack

The goal of this lab is to create a GSM network with internet access and execute calls between our mobile phones.

## Build you LAB

1. Using Docker
2. Install in Debian



- 2 SIM cards provisioned to our network
- 1 BTS
- GSM Core - OSMO
- PBX Asterix
- Internet Connection
